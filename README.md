# traveler's guide

**Traveler's Guide** is a selfhostable dashboard for tracking your Genshin dailies and weekly mission progress. 
It also has support for seeing current banners and events, a simple team builder and a daily farmable materials monitor.

**Full Disclaimer:** Coded with Claude AI.

## Running in Docker
### Clone
Clone this repository:
```
git clone https://github.com/Elderflowa/travelers-guide
cd travelers-guide
```
### Run with compose
```
docker compose up -d
```
